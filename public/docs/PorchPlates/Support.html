Before You Contact Us

To help us assist you more quickly, please include:

  • Your account email (the one you use to sign in)
  • Device information (iPhone model, iOS version)
  • App version (found in Settings > About)
  • Description of the issue with any error messages
  • Screenshots if applicable (helps us understand the problem)
  • Steps to reproduce the issue if possible

Report a Problem

If you've encountered a specific issue, please provide details in your email:

For Order Issues:

  • Order number or reference
  • Seller or buyer name
  • Date of transaction
  • Description of the problem

For Technical Problems:

  • When did the issue start?
  • Does it happen every time or intermittently?
  • Have you tried restarting the app?
  • Is your app up to date?
